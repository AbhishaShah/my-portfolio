Bug Tracker System

Project Overview

Bug tracking and managing system

- This software is mainly connection between QA Testers and Developers as well as designers.
- After Development of particular project fetuare then Project manager create the QA tasks as per modules in online system. In that all project urls, documentations, exception criteria included in that and assigned to the QA members. So automatically QA members get the ideas what they need to check and what not. Besides of that there is a QA bug link. By clicking on that QA bug sheet open in tabular format.
- Filter of different QA bug types like designing base or functional, browser compatibility or Syntax based with different status like open, completed, in review provided.
- And simultaneously this sheet can also be viewed by developer. So at the same time developer can see the bug and solve it in another version of project and can change the status like completed or ignored as exceptions.
-Easy to use and review process, can minimise the bugs and get overall idea of QA Process and timings for Project manager, developers and whole team.

Technology Used: HTML5 CSS3 MYSQL PHP Javascript jQuery jQGrid Bootstrap
